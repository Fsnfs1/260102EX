高三數學衝刺特訓

核心題型與解題模板

260102

教學行為：這是送分題，引導全班齊聲朗讀一遍。 強調：斜率是「縱座標差除以橫座標差」。

考前必背：5大核心公式

1. 立方差公式：
\[ a^3 - b^3 = (a-b)(a^2 + ab + b^2) \]

2. 同底數冪： \[ a^m \cdot a^n = a^{m+n} \]

3. 換底公式： \[ \log_a b = \frac{\log_c b}{\log_c a} \]

4. 斜率與點到直線距離：
\[ k = \frac{y_2 - y_1}{x_2 - x_1}, \quad d = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}} \]

重點提示：強調斜率公式分子分母別寫反。點到直線距離分母是平方和開根號。

基礎概念 (秒殺題)

  • 題目 1 (補集)： \(U=\{1..6\}, A=\{1,3,5\}\)，求 \(\mathcal{C}_U A\)。
    答案：\(\{2,4,6\}\)
  • 題目 2 (奇偶性)： \(f(x)=x^3\) 是什麼函數？
    答案：奇函數
  • 題目 3 (交集)： \(A=\{-3 < x < 5\}, B=\{2 < x < 8\}\)。
    答案：\(\{x \mid 2 < x < 5\}\)
概念辨析：補集是「U有A無」，交集是「重疊部分」。建議畫數軸演示題目3。

解析幾何小問

  • 題目 1 (拋物線)： \(y^2 = 12x\)，求焦點。
    解法：\(4p=12 \Rightarrow p=3\)
    答案：\((3, 0)\)
  • 題目 2 (直線截距)： 點 \(A(-4, 9)\) 在 \(y = -2x + m\) 上。
    解法：代入 \(9 = -2(-4) + m\)
    答案：\(m=1\)
歸納：看到「點在直線上」，機制就是「代入座標」。

統計與多項式

  • 期望值： \(P(X)\) 分別為 0.3, 0.4, 0.2, 0.1 (對應 0,1,2,3)。
    算式：\(0(0.3)+1(0.4)+2(0.2)+3(0.1)\)
    答案：1.1
  • 方差性質： \(D(Y)=1\)，求 \(D(-3Y+5)\)。
    答案：\((-3)^2 \times 1 = 9\)
  • 餘數定理： \(-3x^2+x+4\) 除以 \(x-1\)。
    解法：代入 \(x=1\)
    答案：2
小結：方差性質係數要平方，常數項無效。

橢圓性質：標準方程分析

\[ \frac{x^2}{9} + \frac{y^2}{25} = 1 \]
  • 1. 認準分母： \(25 > 9 \Rightarrow\) 焦點在 \(y\) 軸
  • 2. 參數： \(a=5, \quad b=3\)
  • 3. 計算 c： \(c = \sqrt{a^2 - b^2} = \sqrt{16} = 4\)
  • 4. 結論： 焦點 \((0, \pm 4)\)，離心率 \(e = \frac{4}{5}\)
邏輯順序：先定位置，再算abc。注意 c^2 = a^2 - b^2。

對數方程：解題三部曲

題目：\(\lg(x-1) + \lg(x-2) = \lg(x+2)\)

  1. 合併： \(\lg[(x-1)(x-2)] = \lg(x+2)\)
  2. 去對數： \(x^2 - 3x + 2 = x + 2 \Rightarrow x(x-4)=0\)
  3. 檢驗 (關鍵)：
    • \(x=0\) (捨去，真數小於0)
    • \(x=4\) (成立)
重點提示：必須檢驗真數大於0，否則扣分。

韋達定理

題目：\(x^2 - 5x + 6 = 0\)，根為 \(\alpha, \beta\)

\[ \alpha + \beta = 5, \quad \alpha\beta = 6 \]

求 \(\alpha^2 + \beta^2\)：

\[ (\alpha + \beta)^2 - 2\alpha\beta = 5^2 - 2(6) = 13 \]

目的-方法-結果：直接套公式，避免解方程的繁瑣。

數學歸納法 (MI)

證明：\(1+3+\dots+(2n-1) = n^2\)

  • Step 1 (首步)： 當 \(n=1\)，左=1，右=1，成立。
  • Step 2 (假設)： 設 \(n=k\) 時成立，即 \(1+\dots+(2k-1) = k^2\)。
  • Step 3 (推導)： 當 \(n=k+1\) 時， \[ \text{左} = k^2 + [2(k+1)-1] = (k+1)^2 = \text{右} \]
評分標準：這三步標題寫出來就有大部分分數。

部分分式：數值代入法

將 \(\frac{5x+13}{(x+2)(x+3)}\) 拆解：

\[ 5x+13 = A(x+3) + B(x+2) \]
  • 令 \(x=-3 \Rightarrow -2 = -B \Rightarrow B=2\)
  • 令 \(x=-2 \Rightarrow 3 = A \Rightarrow A=3\)

答案：\(\frac{3}{x+2} + \frac{2}{x+3}\)

技巧：遮蓋法原理，百思不得其解的方程瞬間簡化。

綜合除法

題目：\((2x^3 + 3x - 62) \div (x-3)\)

  • 陷阱： 缺項補 0！係數寫為 \([2, 0, 3, -62]\)
  • 運算口訣： 首項落下，斜乘直加。
商式：\(2x^2 + 6x + 21\)，餘數：1
操作細節：強調「補0」是最大陷阱。

圓的解析幾何：弦長公式

圓 \((x-2)^2 + (y-5)^2 = 169\)，直線 \(3x+4y-1=0\)

  1. 讀數： \(C(2,5), r=13\)
  2. 弦心距 \(d\)： \[ d = \frac{|3(2)+4(5)-1|}{5} = \frac{25}{5} = 5 \]
  3. 弦長： \[ 2\sqrt{r^2 - d^2} = 2\sqrt{169-25} = 24 \]
策略：幾何法優於聯立方程法，計算量小。

70分拿分策略

  • 填空題： 公式背熟，直接得分。
  • 解答題：
    • MI 寫出三步標題。
    • 圓錐曲線寫出 \(a,b,c\)。
    • 方程題必寫「經檢驗」。
